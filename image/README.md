# Glenwood image archive

## Contents

- `official-website/`: images referenced by Glenwood's six property-specific pages (homepage, property overview, bedrooms, twin/family rooms, cottage and public gallery).
- `sources.csv`: provenance for every downloaded file, including the source page, original image URL, alt text, byte size and download status.

## Social account check

As of 12 July 2026, no Facebook or Instagram account could be verified as an official Glenwood Guest House account from the property's website or public search results.

The X account embedded by the website is `@Come2BetwsYCoed`. It is a destination/tourism account, not a Glenwood-owned account, so its images were not mixed into this property archive.

A public directory mentions a Twitter link but did not expose a verifiable account URL in its public page response. Do not attribute social images to Glenwood until the owner confirms the exact handles.

## Usage note

These files are collected for a private redesign demo. Public availability is not the same as a licence to republish. Obtain the property owner's permission before deploying them on a live website. Social-platform and guest-uploaded images require separate permission from their rights holders.
